## Item Management API

## Description
This is a Spring Boot REST API to add and get items.

## Technologies Used
- Java
- Spring Boot
- MySQL
- JPA
- Postman

## How to Run:-
1. Clone the project
2. Open in IntelliJ
3. Configure database in application.properties
4. Run the application

## API Endpoints:-

- ### Add Item
POST /api/items/add-item

- ### Get Items
GET /api/items/1
